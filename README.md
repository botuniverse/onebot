# OneBot

**OneBot** 是一个聊天机器人应用接口标准，旨在统一不同聊天平台上的机器人应用开发接口，使开发者只需编写一次代码即可应用到多种机器人平台。

## 特点

- **简单**：接口简单易懂，可轻松接入
- **兼容性**：兼容原 CQHTTP 插件，零负担迁移
